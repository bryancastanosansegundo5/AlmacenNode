# Almacén Node.js - Backend

Este es el backend de un sistema de gestión de productos y categorías para un almacén, desarrollado en Node.js con Express y MySQL.

## Tecnologías

- Node.js
- Express
- MySQL
- dotenv

## Endpoints principales

- `GET /categorias/all`
- `POST /categorias`
- `PUT /categorias/:id`
- `DELETE /categorias/:id`
- `GET /productos/all`
- `POST /productos`
- ...
